# Directives Not Supported in ModSecurity v3

The following directives exist in ModSecurity v2 but are NOT supported in v3.
They are listed here for reference only and should not be used in CRS rule generation.

- SecCacheTransformations
- SecChrootDir
- SecCollectionTimeout
- SecConnEngine
- SecConnReadStateLimit
- SecConnWriteStateLimit
- SecContentInjection
- SecCookieFormat
- SecCookieV0Separator
- SecDataDir
- SecDisableBackendCompression
- SecGsbLookupDb
- SecGuardianLog
- SecHashEngine
- SecHashKey
- SecHashMethodPm
- SecHashMethodRx
- SecHashParam
- SecInterceptOnError
- SecPcreMatchLimitRecursion
- SecRequestBodyInMemoryLimit
- SecRuleInheritance
- SecRulePerfTime
- SecSensorId
- SecServerSignature
- SecStatusEngine
- SecStreamInBodyInspection
- SecStreamOutBodyInspection
- SecTmpDir
- SecTmpSaveUploadedFiles
